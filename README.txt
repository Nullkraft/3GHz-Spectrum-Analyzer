This is the First set of Documents for the Spectrum Analyser project 19Feb 2018

Goals:
1) Frequency Range: DC-3000 MHz
2) Dynamic Range: 80 dB min; 90 dB typ
3) Operating signal level <0dBm;
   Max RF Input +10dBm DC-48MHz ; +15 dBm 48-3000MHz
4) IF Bandwidths : A:~ 300KHz B:~ 7.5 KHz (Dual simultaneous IF's)
5) Power Supply requirement: +12 VDC at 1 A typ
6) Interface: Serial SPI Bus 3-wire . Clock and Data Common,
   LE signal is specific to the device being programmed.
7) Controllable from any Arduino UNO R3 or better microcontroller/ FPGA.
